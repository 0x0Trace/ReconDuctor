## Screenshots

Screenshots should be placed in the `images/` directory:

- `images/report.png` - Main HTML vulnerability report
- `images/ai-triage.png` - AI triage report
- `images/list-scans.png` - list-scans command output
- `images/gau-results.png` - GAU historical URLs report

### Screenshot Requirements

- Resolution: 1920x1080 or similar
- Format: PNG
- Redact any sensitive information (domains, IPs, credentials)
